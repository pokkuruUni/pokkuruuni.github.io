# ポケモン図鑑クイズメーカー

このリポジトリは、1台のブラウザから複数人で遊べる「ポケモン図鑑説明クイズアプリ」です。
司会者（ホスト）が問題を読み上げ、参加者（ゲスト）が自身のスマートフォン等から回答を送信する、オンラインマルチプレイ機能に対応しています。

## 🌟 主な機能

**問題作成エディタ**: ポケモンの名前を入力するだけで、図鑑説明、分類、タイプ、公式画像を自動で取得・生成します。（手動入力も可能）

**出題モード（ホスト）**: 文字送りエフェクト付きで図鑑説明を読み上げ可能。タイマー機能（自動進行）、参加者のリアルタイム回答確認機能を搭載。

**参加モード（ゲスト）**: 部屋番号を入力して入室。1つのヒントにつき最大3回まで回答可能で、即座に正誤判定が表示されます。

**チーム戦モード**: 参加者を4つのチーム（赤/青/緑/黄）に分け、チーム合計得点で競い合うことができます。
